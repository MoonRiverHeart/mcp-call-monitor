MCP Call Monitor

自动采集、聚合、可视化 OpenCode 会话中所有工具与 MCP 调用数据 — 四维洞察：模型、时间、MCP 种类、工具种类

查看实时数据 配置指南

核心功能

四维聚合

按模型、时间、MCP 服务器、工具种类四个维度聚合调用数据，洞察使用模式与趋势

实时采集

PostToolUse Hook 自动捕获每次工具调用，零配置、零侵入、后台非阻塞

热门排名

实时排名 MCP 服务器、工具、模型调用热度，Top 3 金色高亮，新增项 NEW 标签

增长曲线

ECharts 可视化每小时/每日调用趋势，总调用、成功、错误三线对比

如何使用

  1. 1

    启动 Express 后端

    运行 server 接收 Hook 数据，端口 3210

  2. 2

    配置 Claude Code Hook

    在 .claude/settings.local.json 注册 PostToolUse Hook

  3. 3

    开始使用

    Hook 自动捕获所有工具调用，Dashboard 实时可视化

// .claude/settings.local.json
{
"hooks": {
"PostToolUse": [{
"matcher": "",
"hooks": [{
"type": "command",
"command": "node report-call.js",
"async": true
}]
}]
}
}
